# Plan 03-01: Card Hover Effects, Tag Badge Glow, Transition Coordination, and Component Touch Targets

## Goal

All interactive elements across the site respond with polished, consistent micro-interactions. Post cards have a hover lift + shadow + gold glow border effect. Tag badges glow on hover. Every interactive element uses coordinated 200ms ease-smooth transition timing. All component-level interactive elements have 44px mobile touch targets.

## Requirements

- **CARD-01**: Post cards have hover micro-interaction: subtle translateY lift + shadow emergence
- **CARD-02**: Card border transitions to soft gold glow on hover (not hard gold border) -- must use box-shadow, not border-color
- **CARD-03**: Tag badges have subtle gold glow effect on hover
- **TRNS-01**: All interactive elements have consistent transition timing (150-200ms ease-out)
- **TRNS-02**: Nav links, footer links, and tag badges all use coordinated hover transitions
- **RESP-01**: All interactive elements have minimum 44px touch targets on mobile (component-level elements handled here; page-level back-links handled in Plan 03-03)

## Changes

### File: `app/assets/css/main.css`

**Add `--shadow-card-glow` token to the `@theme` block** (after the existing `--shadow-card-hover` line):

```css
--shadow-card-glow: 0 4px 12px rgba(0, 0, 0, 0.5), 0 0 15px rgba(201, 184, 140, 0.15);
```

This is a compound shadow: the first value is the elevation shadow, and the second value is the gold glow (using the gold hex #C9B88C as RGB values at 0.15 opacity for a soft diffused effect).

No other changes to main.css in this plan. Page transition CSS is handled in Plan 03-02.

### File: `app/components/BlogPostCard.vue`

**Replace the current `<article>` element classes.** Current:

```
bg-dark-surface border border-dark-border rounded-lg p-8 transition-colors hover:border-gold/50
```

Replace with:

```
bg-dark-surface border border-dark-border rounded-lg p-5 sm:p-8 shadow-card transition-all duration-200 ease-smooth hover:-translate-y-0.5 hover:shadow-card-glow hover:border-transparent
```

Key changes:
- `p-8` becomes `p-5 sm:p-8` (RESP-01/RESP-03: proportional mobile padding)
- Add `shadow-card` for resting state shadow
- `transition-colors` becomes `transition-all` (needed for transform + shadow + border)
- Add `duration-200 ease-smooth` for standardized timing
- Remove `hover:border-gold/50` (the old hard gold border approach)
- Add `hover:-translate-y-0.5` (subtle 2px upward lift)
- Add `hover:shadow-card-glow` (compound shadow with gold glow, replaces border glow)
- Add `hover:border-transparent` (hide the border so it doesn't compete with the glow shadow)

**Update the `<h2>` element.** Current:

```
mt-3 text-xl font-semibold tracking-tight text-cream group-hover:text-gold transition-colors
```

Add duration/easing for consistency:

```
mt-3 text-xl font-semibold tracking-tight text-cream group-hover:text-gold transition-colors duration-200 ease-smooth
```

### File: `app/components/TagBadge.vue`

**Update the linked variant (NuxtLink).** Current:

```
inline-block text-xs px-2.5 py-1 rounded-full border border-gold/30 text-gold-dim hover:border-gold hover:text-gold transition-colors
```

Replace with:

```
inline-flex items-center min-h-[44px] sm:min-h-0 text-xs px-2.5 py-1 rounded-full border border-gold/30 text-gold-dim hover:text-gold hover:border-gold/50 hover:shadow-[0_0_8px_rgba(201,184,140,0.2)] transition-all duration-200 ease-smooth
```

Key changes:
- `inline-block` becomes `inline-flex items-center` (for vertical centering with touch target)
- Add `min-h-[44px] sm:min-h-0` (RESP-01: 44px touch target on mobile, natural height on desktop)
- `transition-colors` becomes `transition-all` (needed for shadow)
- Add `duration-200 ease-smooth` for standardized timing
- `hover:border-gold` softens to `hover:border-gold/50` (less aggressive)
- Add `hover:shadow-[0_0_8px_rgba(201,184,140,0.2)]` (subtle gold glow)

**Update the unlinked variant (span).** Current:

```
inline-block text-xs px-2.5 py-1 rounded-full border border-gold/30 text-gold-dim
```

Replace with (visual consistency with flex layout, no hover or touch target needed):

```
inline-flex items-center text-xs px-2.5 py-1 rounded-full border border-gold/30 text-gold-dim
```
