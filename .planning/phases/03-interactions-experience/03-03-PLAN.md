# Plan 03-03: Page-Level Responsive Adjustments

## Goal

Ensure page-level elements (not components, which are handled in Plan 03-01) work flawlessly on mobile: typography scales appropriately, spacing adjusts proportionally, and back-links have adequate touch targets. Header navigation adequacy is confirmed by Plan 03-01's changes.

## Requirements

- **RESP-02**: Typography scale adjusts appropriately for mobile screens
- **RESP-03**: Spacing reduces proportionally on mobile while maintaining breathing room
- **RESP-04**: Header navigation works cleanly on mobile viewports (satisfied by Plan 03-01's gap and touch target changes â€” verified here)

Note: RESP-01 (44px touch targets) for component-level elements is handled in Plan 03-01. This plan handles page-level back-links only.

## Changes

### File: `app/pages/index.vue`

**RESP-03: Adjust post list spacing on mobile.** Current (after Plan 03-02 adds hero glow):

```html
<section class="space-y-8">
```

Replace with:

```html
<section class="space-y-6 sm:space-y-8">
```

Tightens post card spacing from 32px to 24px on mobile.

**RESP-02: Adjust hero subtitle for mobile.** Current (after Plan 03-02):

```html
<p class="relative mt-4 text-lg text-cream-muted max-w-2xl leading-relaxed">
```

Replace with:

```html
<p class="relative mt-3 sm:mt-4 text-base sm:text-lg text-cream-muted max-w-2xl leading-relaxed">
```

Steps subtitle from 18px to 16px on mobile and tightens top margin.

### File: `app/pages/post/[...slug].vue`

**RESP-01: Add touch target on back link.** Current:

```html
<NuxtLink to="/" class="text-sm text-cream-muted hover:text-gold transition-colors">
```

Replace with:

```html
<NuxtLink to="/" class="inline-flex items-center min-h-[44px] sm:min-h-0 text-sm text-cream-muted hover:text-gold transition-colors duration-200 ease-smooth">
```

**RESP-03: Adjust header margins for mobile.** Current:

```html
<header class="mt-8 mb-12">
```

Replace with:

```html
<header class="mt-6 sm:mt-8 mb-8 sm:mb-12">
```

Reduces gap between back-link and header, and header and content, on mobile.

**RESP-02: Post title is already responsive (`text-3xl sm:text-4xl lg:text-5xl`).** No change needed.

### File: `app/pages/tags/[tag].vue`

**RESP-01: Add touch target on back link.** Current:

```html
<NuxtLink to="/" class="text-sm text-cream-muted hover:text-gold transition-colors">
```

Replace with:

```html
<NuxtLink to="/" class="inline-flex items-center min-h-[44px] sm:min-h-0 text-sm text-cream-muted hover:text-gold transition-colors duration-200 ease-smooth">
```

**RESP-03: Tighten post list spacing on mobile.** Current:

```html
<section class="space-y-6">
```

Replace with:

```html
<section class="space-y-4 sm:space-y-6">
```

### File: `app/pages/about.vue`

**RESP-03: Adjust top padding for mobile.** Current:

```html

```

Replace with:

```html

```
